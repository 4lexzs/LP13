# Kubernetes Lernprojekt

## Technologie-Auswahl: Kubernetes

### Begründung:
- Kubernetes ist der Industrie-Standard für Container-Orchestrierung
- Wird von 88% der Unternehmen verwendet die Container nutzen
- Extrem gefragtes Skill im DevOps-Bereich
- Ermöglicht automatisches Scaling und Self-Healing von Anwendungen
- Ich möchte verstehen, wie moderne Cloud-Infrastruktur funktioniert

### Ziele für diese 4 Wochen:
1. Verstehen was Kubernetes ist und warum es wichtig ist
2. Lokales Kubernetes Cluster aufsetzen (Minikube)
3. Erste App deployen
4. Grundkonzepte verstehen (Pods, Services, Deployments)
5. Eigenes kleines Projekt deployen

### Voraussetzungen:
- Docker-Kenntnisse (werde ich parallel lernen)
- Kommandozeile (Terminal)
- Grundlegendes Verständnis von Netzwerken
